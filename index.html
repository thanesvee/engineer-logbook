📱 วิธีติดตั้งเป็นแอปในมือถือ

🤖 Android (Chrome)
1. เปิดไฟล์ด้วย Chrome
2. กดเมนู ⋮ (3 จุด) มุมขวาบน
3. เลือก "เพิ่มไปยังหน้าจอโฮม"
4. กด "เพิ่ม" → ไอคอนจะปรากฏในหน้าจอโฮม
🍎 iPhone (Safari)
1. เปิดไฟล์ด้วย Safari เท่านั้น
2. กดปุ่ม แชร์ (สี่เหลี่ยมมีลูกศรขึ้น) ด้านล่าง
3. เลื่อนลงหา "เพิ่มไปที่หน้าจอโฮม"
4. กด "เพิ่ม" → ไอคอนจะปรากฏในหน้าจอโฮม
หมายเหตุ iPhone: แนะนำ Export ข้อมูลสำรองไว้เป็นประจำ เพราะ Safari อาจล้างข้อมูลถ้าไม่ได้เปิดแอปนานกว่า 7 วัน

💾 Export ข้อมูล

📂 Import ข้อมูล
